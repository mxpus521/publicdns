📚 使用指南

🪟 Windows 10/11

设置 → 网络和互联网 → 更改适配器选项 → 右键网络连接 → 属性 → IPv4属性 → 使用以下DNS服务器地址 → 填入DNS地址

🍎 macOS

系统偏好设置 → 网络 → 高级 → DNS → 点击+号添加DNS地址

🐧 Linux

编辑 /etc/resolv.conf 或 /etc/netplan/01-netcfg.yaml 文件，添加 nameserver 8.8.8.8

📱 Android

设置 → 网络和互联网 → 高级 → 私有DNS → 输入DNS地址

🍏 iOS/iPadOS

设置 → Wi-Fi → 点击网络 → 配置DNS → 使用手动 → 添加DNS服务器

📡 路由器配置

登录路由器管理界面 → 网络设置 → DNS设置 → 填入DNS地址即可让整个网络使用

❓ 常见问题

什么是DNS？

DNS（Domain Name System）是域名系统，它将域名（如 www.example.com）转换为IP地址，使您的设备能够连接到互联网上的网站和服务。

为什么要更改DNS？

更改DNS可以提升网络速度、改进隐私保护、避免DNS污染、获得更好的安全防护等优势。不同的DNS提供商有不同的特点，您可以根据需求选择。

IPv4和IPv6有什么区别？

IPv4是较早的互联网协议，使用32位地址（如 8.8.8.8）；IPv6是较新的协议，使用128位地址（如 2001:4860:4860::8888），支持更多地址。

如何检测DNS是否生效？

使用 nslookup 命令（Windows）或 dig 命令（Linux/Mac）查询DNS解析，或访问 DNS检测网站 查看您当前使用的DNS服务。

是否需要同时填入IPv4和IPv6？

如果您的网络支持IPv6，建议同时配置；如果不支持或不确定，只配置IPv4即可。许多系统允许配置两个IPv4地址和两个IPv6地址。
