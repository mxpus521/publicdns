## 🎯 使用指南

### 配置DNS的方法

#### Windows 10/11
1. 右键点击网络图标 → 打开网络和Internet设置
2. 点击"更改适配器选项"
3. 右键您的网络连接 → 属性
4. 选择"Internet协议版本4(TCP/IPv4)" → 属性
5. 选择"使用下面的DNS服务器地址"
6. 输入首选DNS和备用DNS地址

#### macOS
1. 系统偏好设置 → 网络
2. 选择网络连接 → 高级
3. 点击DNS标签
4. 点击"+"添加DNS服务器地址

#### Linux
编辑 `/etc/resolv.conf` 或 `/etc/netplan/01-netcfg.yaml`：
```
nameserver 8.8.8.8
nameserver 1.1.1.1
```

#### 移动设备
- **Android**：设置 → 网络和互联网 → 高级 → 私有DNS
- **iOS**：设置 → Wi-Fi → 网络详情 → 配置DNS
